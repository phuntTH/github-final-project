# Simple Interest Calculator

A simple Bash script that calculates the simple interest given the principal amount, annual rate of interest, and time period in years.

## Description
This project is a CLI-based tool designed to help users quickly compute simple interest. It takes three inputs from the user, applies the standard formula, and outputs the final calculated interest.

## Input Fields
* **Principal Amount:** The initial sum of money borrowed or invested.
* **Rate of Interest:** The annual interest rate (as a percentage).
* **Time Period:** The duration of the loan or investment in years.

## Formula Used
The script calculates interest using the standard formula:
Interest = (Principal * Rate * Time) / 100
