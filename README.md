# Cars Dealership National Retailer Application

## Project Overview
This repository contains the Full-Stack Capstone Project for the Cars Dealership web application. The platform serves as a responsive national car retailer website in the U.S., allowing users to look up individual dealership branches across states, read historical customer reviews, and submit new reviews with real-time sentiment analysis.

## Tech Stack
* **Frontend:** React, HTML5, CSS3, Bootstrap
* **Backend Microservices:** Node.js (Dealership/Review management), Django (User Authentication), Flask (Sentiment Analysis AI)
* **Databases:** MongoDB (Reviews/Dealerships data), SQLite (User/Car inventory data)
* **DevOps & Cloud:** Docker, Kubernetes, GitHub Actions (CI/CD), IBM Cloud Code Engine
